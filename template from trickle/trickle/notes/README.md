# Kloset - Fashion Rental Marketplace

## Project Overview

Kloset is a fashion rental marketplace connecting renters with sellers of premium designer outfits for special occasions like weddings, festivals, and celebrations.

## Core Value Proposition

**Rent Designer Fashion. Wear Once. Return Easily.**

Users can browse designer outfits, book rental dates, wear the outfit, and return it hassle-free—without purchasing.

## Target Users

**Primary:**
- Women and girls
- Wedding and bridal shoppers
- Festival attendees
- Fashion enthusiasts

**Secondary:**
- Boutique owners
- Fashion designers
- Rental businesses

## Key Features

1. **Browse & Search** - Search hundreds of designer outfits
2. **Book Rental Dates** - Select dates and complete secure payment
3. **Wear & Return** - Receive outfit, wear it, and return easily

## Design Philosophy

- Extremely simple and intuitive
- Mobile-first approach
- Modern, trustworthy, elegant
- No visual clutter
- Original design language (not copying existing platforms)

## Homepage Sections

1. Hero - Large fashion imagery with visual rental flow explanation
2. Shop by Occasion - 6 occasion-based discovery (Wedding, Engagement, Reception, Festival, Party, Birthday)
3. Trending Outfits - Image-first premium product cards
4. How It Works - Visual 3-step process with luxury aesthetic
5. Trust Section - Security and support features
6. Real Stories - Emotional customer testimonials
7. FAQ - Common questions
8. Footer - Minimal elegant footer
